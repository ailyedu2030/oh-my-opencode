# Pitfalls: A股多智能体量化交易系统

## 常见错误

### 1. 数据源问题
- **AKShare限速**: 免费数据可能被限IP → 解决方案: 缓存 + 备用源
- **数据质量问题**: 缺失值/错误值 → 解决方案: 数据清洗 + 验证

### 2. Agent设计问题
- **Agent分工不清**: 职责重叠 → 清晰定义每个Agent的角色
- **决策分歧**: Agent意见不一致 → 投票机制 + 置信度加权

### 3. 方差缩小误区
- **简单平均**: 不是最优 → 使用置信度加权
- **过度拟合**: 因子过多样本内好 → 样本外验证

### 4. 风控漏洞
- **无止损**: 单日亏损无限 → 强制止损线
- **仓位集中**: 单票仓位过高 → 分散度控制

## 预防策略

| 陷阱 | 预警信号 | 预防 |
|-----|---------|-----|
| 数据失效 | 连续空值 | 备用数据源 |
| Agent分歧 | 投票4:1 | 降低置信度 |
| 过拟合 | 样本外IC下降 | 滚动验证 |
| 风控失效 | 连续亏损 | 熔断机制 |
